Perceptron

Classification Linear Est. 1958

The simplest linear classifier — updates weights based on misclassified samples using an online learning rule. Historically the first neural network algorithm.

Epoch:10

Overview

The Perceptron, introduced by Frank Rosenblatt in 1958, is the simplest possible linear classifier. It processes samples one at a time: if a sample is correctly classified, no update is made; if misclassified, the weight vector is adjusted in the direction of the correct classification. For linearly separable data, the Perceptron is guaranteed to converge in a finite number of steps.

When to Use

  • Data is linearly separable (or nearly so)
  • You need an online learning algorithm
  • As a building block for understanding neural networks
  • You need the absolute fastest possible classifier

Pros

  • Extremely simple and fast
  • Online learning (processes one sample at a time)
  • Convergence guaranteed for linearly separable data
  • Foundation of neural network theory

Cons

  • Only works for linearly separable data
  • No probability estimates
  • Solution depends on initialization and data order
  • No convergence guarantee for non-separable data
  • No regularization

How It Works

  1. Initialize weights $w$ to small random values (or zeros).
  2. For each sample $(x_i, y_i)$ where $y_i \in \{-1, +1\}$: compute $\hat{y} = \text{sign}(w \cdot x_i + b)$.
  3. If misclassified ($\hat{y} \neq y_i$): update $w \leftarrow w + \eta \, y_i \, x_i$ and $b \leftarrow b + \eta \, y_i$.
  4. Repeat until no misclassifications or max epochs reached.

Use the epoch slider to see how the decision boundary evolves during training. Early epochs show a poor boundary that gradually rotates into the correct position.

Key Equations

Perceptron Update Rule

$$w^{(t+1)} = w^{(t)} + \eta \, y_i \, x_i \quad \text{if } y_i (w^{(t)} \cdot x_i) \le 0$$

Convergence Theorem

If the data is linearly separable with margin $\gamma$ and all $\|x_i\| \le R$, the perceptron converges in at most $$T \le \left(\frac{R}{\gamma}\right)^2$$ updates.

Decision Function

$$\hat{y}(x) = \text{sign}(w \cdot x + b)$$

scikit-learn

Python
from sklearn.linear_model import Perceptron
from sklearn.metrics import accuracy_score

model = Perceptron(max_iter=1000, eta0=1.0, random_state=42)
model.fit(X_train, y_train)

y_pred = model.predict(X_test)
print(f"Accuracy: {accuracy_score(y_test, y_pred):.4f}")
print(f"Weights: {model.coef_}")
print(f"Bias:    {model.intercept_}")
