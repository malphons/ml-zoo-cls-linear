Ridge Classifier

Classification Linear Est. 1970

Treats classification as a regression problem with L2 penalty, assigning classes based on the sign of the prediction. Extremely fast due to the closed-form Ridge solution.

alpha:1

Overview

The Ridge Classifier converts classification into a regression problem. It encodes targets as -1/+1 (or one-hot for multiclass), solves the Ridge regression problem $(X^TX + \alpha I)^{-1}X^Ty$, and classifies based on the sign of the prediction. Because it uses a closed-form solution, it is much faster than iterative methods like Logistic Regression.

When to Use

  • You need a very fast linear classifier
  • High-dimensional data (e.g. text classification with TF-IDF)
  • You don't need calibrated probabilities

Pros

  • Closed-form solution — extremely fast
  • L2 regularization prevents overfitting
  • Works well in high dimensions
  • Scales to large datasets

Cons

  • No probability estimates (only hard predictions)
  • Linear decision boundary only
  • MSE loss not ideal for classification
  • Sensitive to alpha parameter

How It Works

  1. Encode class labels as $y \in \{-1, +1\}$ (binary) or one-hot vectors (multiclass).
  2. Solve the Ridge regression problem: $\hat{w} = (X^TX + \alpha I)^{-1}X^Ty$.
  3. Classify new point: $\hat{y} = \text{sign}(w \cdot x + b)$.

Adjust the alpha slider to see how regularization affects the decision boundary. Higher alpha shrinks coefficients toward zero, potentially changing the boundary orientation.

Key Equations

Ridge Solution

$$\hat{w} = (X^TX + \alpha I)^{-1}X^Ty$$

Classification Rule

$$\hat{y}(x) = \text{sign}(\hat{w}^T x + \hat{b})$$

Loss Function

$$\mathcal{L}(w) = \|y - Xw\|_2^2 + \alpha \|w\|_2^2$$

Unlike logistic regression (cross-entropy loss), Ridge Classifier minimizes squared error.

scikit-learn

Python
from sklearn.linear_model import RidgeClassifier
from sklearn.metrics import accuracy_score

model = RidgeClassifier(alpha=1.0)
model.fit(X_train, y_train)
y_pred = model.predict(X_test)
print(f"Accuracy: {accuracy_score(y_test, y_pred):.4f}")
