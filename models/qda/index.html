Quadratic Discriminant Analysis

Classification Linear Est. 1936

Like LDA but allows each class its own covariance matrix, producing quadratic (curved) decision boundaries that can capture more complex class separations.

Overview

Quadratic Discriminant Analysis (QDA) extends LDA by allowing each class to have its own covariance matrix. When classes have different spreads or orientations, QDA's quadratic decision boundary fits the data better than LDA's linear one. The trade-off is that QDA estimates more parameters (one covariance per class), requiring more data.

When to Use

  • Classes have clearly different covariance structures
  • Sufficient samples per class to estimate separate covariances
  • Linear boundaries are inadequate

Pros

  • Captures different class shapes and orientations
  • Quadratic boundaries more flexible than linear
  • Closed-form solution
  • Provides class probabilities

Cons

  • More parameters to estimate than LDA
  • Can overfit with small samples
  • Still assumes Gaussian classes
  • Covariance inversion can be unstable

How It Works

QDA works like LDA but fits a separate covariance matrix for each class. The discriminant function becomes quadratic in $x$ because the covariance matrices don't cancel.

  1. Estimate per-class means $\mu_k$ and covariance matrices $\Sigma_k$.
  2. Compute the quadratic discriminant function for each class.
  3. Classify by choosing the class with the highest discriminant value.

Toggle "Compare LDA" to see how the linear boundary differs from QDA's curved boundary. Notice the QDA boundary curves to accommodate the different class shapes.

Key Equations

QDA Discriminant Function

$$\delta_k(x) = -\frac{1}{2}\log|\Sigma_k| - \frac{1}{2}(x - \mu_k)^T\Sigma_k^{-1}(x - \mu_k) + \log\pi_k$$

Decision Boundary

$$\delta_j(x) = \delta_k(x) \implies x^T A x + b^T x + c = 0$$

This is a quadratic equation in $x$, yielding elliptical, parabolic, or hyperbolic boundaries.

LDA vs QDA

LDA: shared $\Sigma$ → linear boundary, $p(K-1) + K \cdot p$ parameters.
QDA: per-class $\Sigma_k$ → quadratic boundary, $K \cdot p(p+1)/2$ additional parameters.

scikit-learn

Python
from sklearn.discriminant_analysis import QuadraticDiscriminantAnalysis
from sklearn.metrics import accuracy_score

model = QuadraticDiscriminantAnalysis()
model.fit(X_train, y_train)
y_pred = model.predict(X_test)
print(f"Accuracy: {accuracy_score(y_test, y_pred):.4f}")
